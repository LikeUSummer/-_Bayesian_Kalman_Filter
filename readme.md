## 各脚本功能

`main.py` 仿真实验的入口脚本

`filters.py` 用精简统一的形式组织和实现了各种贝叶斯非线性滤波器

`models.py` 用统一形式描述各种算例模型

`data.py` 根据算例模型预生成仿真数据，存储为 numpy-npz 文件，可加速仿真过程

`rs.py` 粒子滤波所需的几种重采样算法

`ps.py` 投影统计算法

`kf.py` 实现原始卡尔曼滤波器，它独立于 filters.py 的框架
